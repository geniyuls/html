pointer-events 속성은 마우스 포인터와 요소간의 상호 작용에 관한 설정을 위한 속성이다. 기본값 :auto 값 - auto : 상호작용 가능 - none : 상호작용 불가능 user-select 속성은 사용자의 텍스트 선택에 대한 설정을 위한 속성이다. 기본 값 : auto 값 -auto : 브라우저가 상황에 따라 알아서 판단한다. -all : 요소의 내용이 한번에 선택된다./ -contain : 선택의 범위가 요소를 벗어날 수 없음. 단, 일부 입력 태그에만 적용될 수 있음 -none : 요소의 내용을 선택할 수 없음 -text : 요소의 내용을 선택할 수 있음 z-index 속성은 Z축으로의 배치 순서를 설정하기 위한 속성이다. 값이 클수록 사용자에게 가까이 배치된다. (화면 위로 올라온다.) 단, position 속성 값이 static인 요소는 해당 속성을 무시한다. 기본 값: 0 값 :정수 transition-delay 속성은 트랜지션 발생 시작까지 지연시킬 시간을 지정하기 위한 속성이다. 기본 값: 0 값: 시간 값 transition-duration 속성은 해당 요소에 특정 상황에 따라 변화하는 속성에 대해 해당 속성이 변화하는데 걸리는 시간을 지정하기 위한 속성이다. 값: 시간 값 transition-property 속성은 트랜지션의 적용을 받을 CSS 속성으로 지정하기 위한 속성이다. 해당 속성에 명시되지 않은 CSS 속성은 트랜지션의 효과를 적용받지 않는다. 기본 값 : all 값 -all : 모든 가능한 속성에 트랜지션을 적용한다. -none : 트랜지션을 적용하지 않는다. -속성 이름 : **중간 값이 없는 속성 혹은 값 간에 중간 값이 없는 경우 트랜지션 대상이 아님에 유의한다.** 형태 -[all|none|속성 이름,...] transition-timing-function 속성은 트랜지션이 적용되는데 있어, 시간의 흐름에 따른 변화 완료율을 계산하는 함수를 지정하기 위한 속성이다. 기본 값 : ease 값 -ease -ease-in -ease-in-out -cubic-bezier 함수
